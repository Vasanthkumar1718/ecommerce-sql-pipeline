# E-Commerce Revenue Analysis Pipeline

## Project Overview

This project analyzes e-commerce sales data using SQL.

## Business Questions

1. Which city generates the highest revenue?
2. Which products generate the highest revenue?
3. Who are the top customers?
4. How does revenue change month by month?

## Database Schema

customers
products
orders

## Pipeline Flow

customers
      ↓
orders
      ↓
products
      ↓
sales_data
      ↓
business_reports

## Reports Generated

1. City Revenue
2. Product Revenue
3. Customer Ranking
4. Monthly Revenue

## SQL Skills Used

- Joins
- Views
- Aggregations
- Window Functions
- Date Functions

## Key Insights

- Chennai generated the highest city revenue.
- Laptop generated the highest product revenue.
- Vikram is the highest-value customer.
- June generated the highest monthly revenue.
